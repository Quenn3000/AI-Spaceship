# Qu'est ce que le projet
Ce projet est une expérience personnelle pour apprendre les algorithmes génétiques, datant de 2021 (donc l'anglais est terrible je crois).
Il a été codé en Python3, sans LLM

## Comment le faire fonctionner
Une fois que le repo a été clonné, il faut installer les dépendances avec
``` terminal
python3 -m venv .venv
source ./.venv/bin/activate
python3 -m pip install -r requirements.txt
```
puis vous pouvez le lancer avec
``` terminal
python3 main.py
```
Le seul input qui sera demandé correspond au nombres d'IAs qu'il y aura dans la génération 0
